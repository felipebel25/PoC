# eBook Marketplace

This is a simple React + Vite app showcasing a simulated eBook marketplace with 3 books, featuring:

- **Homepage**: Browse all available books with cover, author, price, and quick links to details.
- **Book Detail Page**: See full details about any book in the database.
- **Navigation**: Smooth routing between home and book pages (via React Router).

## Dummy Books
Three example books are included with images, titles, descriptions, and authors.

## Run the Project

To run locally:

```bash
npm install
npm run dev
```
Open the dev server URL in your browser (shown in the console).
